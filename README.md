# accountTracker
Web app created for tracking finances for each client. Technologies used: HTML, Bootstrap CSS, JavaScript.

## Features
- Creating new client account
- Adding or subtracting from the sum they owe
- View transaction history for each account
- Edit client names
- Delete accounts
- Search for specific accounts
- Local storage persistence (accounts saved automatically)

## Installation
1. Clone the repository
2. Open `index.html` in your browser

## How to Use
1. Click "Create Account" to add a new client
2. Enter client name (first and last) and initial amount owed
3. Use the **+** button to add money owed
4. Use the **-** button to subtract/pay money
5. Click "View Transactions" to see transaction history
6. Click "Edit" to change a client's name
7. Click "Delete Acc" to remove a client
8. Use the search bar to find specific clients

## Technologies Used
- HTML5
- CSS3 (Bootstrap 5)
- Vanilla JavaScript (ES6 Classes)
- Local Storage API
